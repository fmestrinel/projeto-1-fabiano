git init --------------------------------------------------------------	inicializa um novo repositorio
git reflog ------------------------------------------------------------	verifica as versoes
git push --------------------------------------------------------------	envia as atualizações para a nuvem na branch atualmente ativa
git add . -------------------------------------------------------------	adiciona os arquivos atuais ao próximo commit
git status ------------------------------------------------------------	verifica o status atual do repositório
git pull --------------------------------------------------------------	tras atualizações do servidor para a maquina
git commit -m "mensagem do commit" ------------------------------------	cria um novo commit com uma mensagem(breve descrição da alteração que esta sendo feita)
git branch ------------------------------------------------------------	permite listar e ver qual branch esta ativa atualmente
git checkout nome da branch -------------------------------------------	permite mudar para uma nova branch
git checkout -b "nome da branch de origem" "nome da nova branch" ------	permite mudar e ccriar uma nova branch com base em outra
git merge "branch a receber merge" ------------------------------------	permite fazer o merge da branch ativa atualmente com outra branch
git push origin master ------------------------------------------------	Envia o projeto atualizado para a nuvem

******unir as branchs********
1. git pull da branch principal
2. gerar a nova branch a partir da branch principal
3. trabalhar e adicionar novas funcionalidades da nova branch que criou.
4. Finalizar o trabalho na branch temporária
5. Git checkout na branch principal 
6. Git pull 
7. Mergiar(unir) o código da branch temporário com a branch princial (depois testar)
8. Git push da branch principal
